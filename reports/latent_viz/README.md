on the **TUAB eval split**, coloured by clinical label (normal vs abnormal). Latent dim = 256.

- **Encoder:** `capB_nojit` = Conv 1.35M (`hidden=96, depth=4`), VICReg + spectral-0.1 + exact
  corruption, `inv=1`, no scale-jitter, 20 epochs, **seed 0** — the tuned model (per-rec 0.824 / per-win 0.755).
- **Generated by** [`examples/eeg/latent_viz.py`](../../examples/eeg/latent_viz.py):

## Honest read
The latent is **softly, not cleanly, separable**:
- **Per-window** (UMAP/t-SNE): normal and abnormal sit in **one intermixed manifold** with only a soft
  density gradient (abnormal concentrates to one side) — matches the moderate per-window BAcc ≈ 0.76.
- **Per-recording** (pooled): the gradient sharpens — normal concentrates in one region, abnormal in
  another — but they still overlap substantially — matches per-recording BAcc ≈ 0.82.

So the SSL representation encodes a **real but weak/non-linear** normal-vs-abnormal axis: enough for a
linear probe to reach ~0.82 per-recording, but far from cleanly clustered. PCA (linear) shows the
least structure; t-SNE/UMAP reveal the soft manifold gradient.
